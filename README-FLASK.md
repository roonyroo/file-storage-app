# 🗂️ Flask 1xf1 File Storage App

**Python Flask web application** with a 1xf1 button for instant text file storage.

## 🚀 Deploy to Production

### Render.com Settings:
- **Repository:** `roonyroo/file-storage-app`
- **Branch:** `flask-python` 
- **Build Command:** `pip install -r requirements.txt`
- **Start Command:** `python app.py`
- **Environment:** `Python 3`

### Railway.app:
- Auto-detects Python from `requirements.txt`
- Select branch: `flask-python`

## 🛠️ Local Testing

```bash
git clone https://github.com/roonyroo/file-storage-app.git
cd file-storage-app
git checkout flask-python
pip install -r requirements.txt
python app.py
```

Visit: `http://localhost:5000`
